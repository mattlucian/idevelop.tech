## Content Organization Notes

### URL Structure Pattern
```
/                                          # Home
/software-engineering                      # Level 1
/software-engineering/backend              # Level 2
/software-engineering/backend/languages    # Level 3
/software-engineering/backend/languages/java  # Level 4
```

### File Naming Convention
- Lowercase with hyphens
- Descriptive names matching page titles
- Consistent across all levels

### Content Structure
Each file includes:
- Breadcrumb navigation
- Page title and subtitle
- Key concepts/technologies with experience levels
- Links to child pages
- Context section with real-world implementation details
